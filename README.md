# Tyler Szakacs

I build websites, native iOS products, local-first automation tools, MCP servers, and the systems around them.

Most of my day-to-day work lives in private product and client repos. I still keep a growing part of the tooling open source. Private work is named here for context. Only public repos are linked.

## What I'm Building

- AskVero: a private SwiftUI iOS app, its web/services layer, feedback workflow, OSS eval lab, and supporting archive/migration repos.
- Preflight: a private App Store review-readiness platform, with public install/distribution tooling.
- Pallium: a private iOS notes app focused on fast capture and AI-assisted organization.
- Websites, landing pages, and web platforms across private repos for Szakacs Media, GNA Works, Marpe Nutrition, and the Marpe Detox Guide.
- Product experiments, healthcare prototypes, and internal systems across Orion Clinical Intelligence, Artrio, Plaid tooling, workflow automation, SWE evals, and ops repos.

## Open Source

### MCP Servers And Local Automation

- [safari-mcp](https://github.com/tszaks/safari-mcp) - Local macOS Safari MCP server with browser control, extraction, and screenshots.
- [reminders-mcp](https://github.com/tszaks/reminders-mcp) - Local Apple Reminders MCP server with reminder and list management.
- [icloud-calendar-mcp](https://github.com/tszaks/icloud-calendar-mcp) - Local iCloud Calendar MCP server with event CRUD.
- [apple-notes-mcp](https://github.com/tszaks/apple-notes-mcp) - Apple Notes MCP server for local note management.
- [multimodal-imessage-mcp](https://github.com/tszaks/multimodal-imessage-mcp) - iMessage MCP server with search and attachment handling.
- [gmail-multi-inbox-mcp](https://github.com/tszaks/gmail-multi-inbox-mcp) - Multi-account Gmail MCP server with built-in OAuth onboarding.
- [gmail-mcp-server](https://github.com/tszaks/gmail-mcp-server) - Gmail MCP server for Claude Code workflows.
- [app-store-connect-mcp](https://github.com/tszaks/app-store-connect-mcp) - App Store Connect MCP server and release automation tooling.

### Developer Tools And Workflow Infrastructure

- [codex-memory](https://github.com/tszaks/codex-memory) - CLI-first repo memory for risk, co-change, and decision context.
- [codex-sessions](https://github.com/tszaks/codex-sessions) - Small CLI for seeing which Codex sessions are live on your machine.
- [codex-discord-bridge](https://github.com/tszaks/codex-discord-bridge) - Discord-to-Codex bridge with draft pull request workflows.
- [sourcekit-lsp-marketplace](https://github.com/tszaks/sourcekit-lsp-marketplace) - Claude Code plugin that adds Swift and Objective-C support via SourceKit-LSP.

- [homebrew-tap](https://github.com/tszaks/homebrew-tap) - Homebrew tap for Szakacs Media command-line tools.
- [homebrew-preflight](https://github.com/tszaks/homebrew-preflight) - Homebrew tap for installing and updating the Preflight CLI.
